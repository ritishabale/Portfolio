# 3D Portfolio Website - Development Instructions

## Project Overview
Modern, futuristic 3D portfolio website with glassmorphism, neon accents, and interactive elements.

## Tech Stack
- Next.js 14 with TypeScript
- React Three Fiber for 3D graphics
- Tailwind CSS for styling
- Framer Motion for animations
- GSAP for advanced scroll animations
